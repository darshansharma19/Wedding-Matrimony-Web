Installion

Follow these easy steps to install the template from scratch:

  1. Connect to your website with an Ftp client (eg. FileZilla)
  2. Upload the contents of the whole /Template folder to your desired root folder
  3. Once uploaded, edit the template files according to your needs, than save
  4. Refresh your browser in order to see the modifications

Editing

To edit the files in this template, you need a text / code editor software installed on your computer. Below, there are some popular free code editor softwares:

Tutorials A bit about the usuage of the advanced functions.

How to change the website Logo

  1. Make you logo with size like 250px X 50px
  2. Copy and paste into images folder
  3. open index.html and find this class ic-logo
  4. Then change your logo here.

How to add new Menu?

  1. Open any .html file then search this class name .bl
  2. Then copy this code(<li><a href="#"></a></li>) and paste into inside the ul

How to add new option in search drop-down?

  1. Open the index.html and find the class name class="chosen-select"
  2. You can copy the option tag and past the same place but you need to change the value of the option tag.

How to chage the button links

  1. All "buttons" or "a" tags using to navigate the other pages.
  2. "a" tage link update <a href="#">link text here<a>
  3. Just link your html page instead of # tag value.

How to change the button color?

  1. Open css/style.css file. All styles updated in style.css file.
  2. Two major button colors using this template. You can find on --cta-pink1 and --cta-pink1h here you can change the color code values.

Meta tag values updates

  1. meta tag help you to index your website on Search engines like Google, Yahoo, Bing and more.
  2. Open the .html files and change the content values of meta tags like meta keywords, meta descriptions and page title.

How to change the footer Phone number and Email id?

  1. Open the .html file
  2. And search the class name foot-supp
  3. Then change the Phone number and Email id here.

How to change Copyrights name?

  1. Open the .html file and search the class name class="cr".
  2. Then start chnage your Copyrights name here.

How to change lead or enquiry receice email id?

  1. Open the mail folder and openmail-contact.php or mail-reservation.php file search the variable name $mail_to = "info@example.com"; then update your email id.
